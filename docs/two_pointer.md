# Two Pointer

- sorted array에서 합이 target이 되는 두 원소를 구하기 위해 two pointer를 쓸 수 있다.
- l, r을 각각 왼쪽, 오른쪽 양 끝단에서 움직인다.
- https://leetcode.com/problems/two-sum-ii-input-array-is-sorted/

## 언제 쓰나
- 정렬된 배열에서 \"합/차\" 조건을 만족하는 쌍을 찾을 때.
- 배열을 분할하거나, 중복 제거/길이 최소화 같은 문제에서 양 끝을 좁힐 때.

## 기본 패턴
1) 조건이 작으면 l++, 조건이 크면 r-- 처럼 한 쪽만 이동.\n
2) 답을 찾은 뒤 중복을 건너뛰어야 하는 경우가 많음.

## 주의점
- 정렬이 필요하면 먼저 정렬을 하고 인덱스 대신 값 기반으로 생각.\n
- 정렬이 불가능하면 슬라이딩 윈도우/해시를 우선 고려.
