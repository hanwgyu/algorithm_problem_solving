# BFS

- 뭔가 퍼져나가고, 최소의 지점을 구할 때 사용. (엣지의 길이가 주어지지 않을 때)
- 엣지의 길이가 주어지고 최소 경로나 MST를 찾을 때는 다른 알고리즘 사용.

## Multi-source BFS
- 여러 지점으로부터 특정 위치까지의 최소의 지점을 구해야 할 때는 해당 여러 지점을 한번에 큐에 넣어놓고, 해당 지점까지 먼저 도달하게 하면 된다.
- 0, 1로 이루어진 Array의 각 위치에서 0까지의 최소의 거리들을 구하는 문제
    - https://leetcode.com/problems/01-matrix/
    - 0들을 모두 큐에 넣어놓고, 한번 1에 방문하면 재방문하지 않게 하면 된다.
